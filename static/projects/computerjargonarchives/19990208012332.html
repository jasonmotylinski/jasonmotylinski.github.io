Computer Jargon

I recently went shopping for PCs and found that pre-configured machines were over-priced and did not provide the features I was looking for. So I have created this web page to help readers construct a cheap and decent machine. My goal is to create the best PC for the price. I recently priced out this machine:

If you feel that you can price out a better machine I challenge you to it. Please send the results. I hope this site can become a resource for people building their own machines. I do have some stipulations to submissions. First, please use a current processor like Intel or AMD. I don't want to see WinChips or P233s. I know, I know, Cyrix has got a good thing going but until the WinChip can *prove* itself with the big boys I don't think that Cyrix is a viable competitor. Secondly, don't use cheap products. On items like floppy drives generic pieces can be used but please stick to quality products. I would like you to build a machine that YOU would want to use. Also, include the dealer where you can obtain the parts and the link to the product so I can verify the price.
